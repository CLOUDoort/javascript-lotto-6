# 기능 구현 목록

- [x] 사용자로부터 금액을 입력 받는다.
- [x] 금액 예외 처리.
  - [x] 숫자만 입력 가능.
  - [x] 1000원 미만, 100_000원 초과 불가능
  - [x] 1000원 단위로 입력 받는다. 나머지는 예외 처리.
- [] 입력받은 금액으로 살 수 있는 로또 개수 결정.
- [] 개수만큼 로또 생성.
  - [] 1~45까지의 수 범위에서 중복되지 않은 6자리 숫자 생성
- [] 생성한 로또 번호 출력.
  - [] 오름차순으로 정렬하여 출력.
- [] 당첨 번호와 보너스 번호 생성.
  - [] 중복되지 않은 6개의 숫자.
  - [] 중복되지 않은 1개의 보너스 숫자.
- [] 당첨 번호와 보너스 번호 출력.
- [] 각 로또 번호를 당첨 번호와 보너스 번호를 비교
- [] 비교한 결과와 수익률을 계산하여 출력.
